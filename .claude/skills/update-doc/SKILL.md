# Update Doc

Use this skill when the user wants to replace or update a documentation file with user-supplied text.

1. Identify the target file from the user's request.
2. Print the current contents of the target file.
3. Ask the user for the replacement text.
4. Show the proposed replacement and ask for explicit confirmation before editing.
5. Apply only after explicit confirmation.
6. Print the resulting file contents.
