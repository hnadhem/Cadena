# Research

This file is a pointer, not implementation guidance.

Historical market, behavior, and early technical research has moved to `docs/archive/research.md`. That archive includes obsolete technical recommendations such as backend sync, React Query, `date-fns`, Supabase, and a draft data model that do not match the current local-first Expo/SQLite architecture.

Use these current sources instead:

- `AGENTS.md` - shared project and agent guidance.
- `CLAUDE.md` - Claude Code adapter that imports shared guidance.
- `schema_consolidated_13.html` - canonical v13 data-shape reference.
- `docs/app-layer-rules.md` - business rules and product behavior not encoded by schema.
- `docs/schema-errata.md` - known schema documentation ambiguities.
- The codebase - source of truth for implemented architecture.
